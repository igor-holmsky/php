# Синтаксис PHP

PHP-код встраивается в HTML с помощью специальных тегов:
```php
<?php
  // PHP-код здесь
?>
```

Также возможны альтернативные формы (чаще в шаблонах), но рекомендуется использовать `<?php ... ?>`.

## Комментарии
```php
// Однострочный комментарий
# Тоже однострочный комментарий
/*
  Многострочный комментарий
*/
```
